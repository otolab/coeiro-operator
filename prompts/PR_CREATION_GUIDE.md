# PR作成ガイド

Claude Codeを使用してPRを作成する際の注意事項とチェックリストです。

## PR作成時の基本フロー

1. **作業ブランチの作成**
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **テストの実行**
   ```bash
   npm run build
   npm test
   npm run test:e2e
   npm run type-check
   ```

3. **コミット前の確認**
   - 不要なデバッグコードが残っていないか
   - console.logが本番コードに含まれていないか
   - TODOコメントは適切か

4. **PR作成時の説明**
